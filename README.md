# ChristBASHTree

You know, a Christmas tree on Bash :)

# Usage

Git clone and execute:

```
git clone https://github.com/6am600/ChristBASHTree.git && cd ./ChristBASHTree && bash tree-EN.sh
```

__Enjoy!__
